Frame is a notetaking desktop and web application designed to seamlessly work with various levels of data analysis, extraction, and generation. Its aim is to eventually provide near instantaneous access for users to massive amounts of collected industry / domain knowledge, through a conversational interface and recursive data structure for storing entries.
